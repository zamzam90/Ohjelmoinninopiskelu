Lasketaan sanalle kirjainpisteet:

  • A, E, I, N, S, T = 1p
  • O, Ä, K, L = 2p
  • U, M = 3p
  • Y, H, J, P, R, V = 4p
  • Ö, D = 7p
  • B, F, G = 8p
  • C = 10p
  • loput kirjaimet = 12p
